## 備品管理システム

### 概要

このシステムでは、社内で扱う備品の在庫管理を行うことができます。
備品の登録、編集、削除、在庫調整を行うことができ、各備品操作のログを見ることができます。

### 主な機能
* ログイン・ログアウト機能
* 備品一覧画面
* 備品登録、編集、削除機能
* 備品在庫管理機能(持出、入庫)
* 備品検索機能
* 備品操作ログ

### 開発環境

    php 8.2.4
    laravel 5.5.2
